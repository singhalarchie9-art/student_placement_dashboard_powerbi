🎓 Student Placement Dashboard | Power BI

📌 Project Overview

This Power BI dashboard analyzes student placement data to uncover insights into academic performance, placement trends, salary packages, and factors influencing successful placements. The dashboard enables interactive analysis using filters, KPIs, and visualizations.

🎯 Objectives
Analyze overall placement performance
Compare placement rates across branches and college tiers
Identify factors affecting placements
Explore salary package distribution
Evaluate student skills and academic performance

📈 Key Insights

Students completing more internships and projects show stronger placement outcomes.
Coding and communication skills positively influence placement success.
Branch and college tier impact placement rates.
