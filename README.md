🎓 Student Placement & Career Analytics Dashboard

An interactive Power BI dashboard designed to analyze student placement performance, academic metrics, career readiness, and skill development. This project transforms raw student data into actionable insights through dynamic visualizations and KPI reporting.

📌 Project Overview

The Student Placement & Career Analytics Dashboard helps educational institutions analyze placement trends and identify factors influencing student success.

The dashboard provides an interactive experience using slicers, KPI cards, charts, and drill-down analysis to monitor placement performance across departments, college tiers, and student skill metrics.

🎯 Objectives
Analyze overall placement performance
Monitor branch-wise placement statistics
Evaluate student academic performance
Assess career readiness indicators
Compare technical and soft skills
Support data-driven placement decisions

📈 Key Insights
Placement Rate reached 78.80%.
CSE students achieved the highest placement numbers.
Students with internships generally secured better placement packages.
Strong GitHub portfolios and higher project counts were associated with improved average packages.
Coding performance showed a positive relationship with placement outcomes.
